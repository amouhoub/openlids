Overview

LInked Data Services (LIDS) denote the integration of data providing services and linked data, which leads to data silos that are opened up to the Web of Data. LIDS define interface conventions that are compatible to linked data principles and are defined by lightweight formal model that enables the automatic creation of LIDS interfaces and the automatic integration of links to LIDS in existing data sets.
