Linked Data Services (LIDS)

Linked Data Services (LIDS) are an approach to open up data providing services to the Web of Data. See also http://openlids.org/

The vocabulary currently consists of two terms: LIDS and their descriptions.

LIDS
Type: Class
URI:http://openlids.org/vocab#LIDS
Label:LInked Data Service (LIDS)
Comment:A data providing service adhering to the Linked Data Service (LIDS) methodology.
 
description
Type: Property
URI:http://openlids.org/vocab#description
Label: description
Comment:A description of a LIDS using a SPARQL CONSTRUCT query
