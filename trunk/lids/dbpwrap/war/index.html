DBpedia University LIDS

Welcome to the DBpedia University Linked Data Service (LIDS). The service returns student numbers for universities from DBpedia, when given a homepage. The interface is for use with Linked Data.

Currently we support the following services:

DBPediaStudNR
Input: homepage of university Output: RDF with the number of students E.g. /DBPediaStudNR?homepage=http%3A%2F%2Fwww.cmu.edu#university.
